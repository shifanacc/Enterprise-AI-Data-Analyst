#  Overview

Enterprise AI Data Analyst is an AI-powered business analytics platform developed using **Transformer-based Deep Learning**, **Natural Language Processing**, and **Business Intelligence** techniques.

The application enables users to upload business datasets and simply ask questions like:

> **"Show monthly sales"**

or

> **"Who are the top customers?"**

The system automatically understands the query, performs business analysis, generates visualizations, provides AI insights, and allows exporting reports.

#  Installation

### Clone Repository

```bash
git clone https://github.com/USERNAME/Enterprise-AI-Data-Analyst.git
```

### Navigate

```bash
cd Enterprise-AI-Data-Analyst
```

```bash
pip install -r requirements.txt
```

### Run

```bash
streamlit run analyst.py
```

<details>

<summary><b> How It Works</b></summary>

1. Upload a CSV or Excel dataset.
2. Dataset is normalized automatically.
3. User asks a business question.
4. Transformer predicts the intent.
5. Analytics engine processes the query.
6. Charts and KPIs are generated.
7. AI Insights are displayed.
8. Reports can be exported as PDF, CSV, or Excel.
